uint32_t EI_API_ADP_isHwSettingEnabled ( T *  pAdp_p,
bool *  pHwConfigEnabled_p 
)

This function returns the hardware settings.

The function returns true if the hardware settings are enabled. Otherwise false.

Parameters
[in]pAdp_pPointer to the adapter.
[out]pHwConfigEnabled_pPointer to a boolean that is set to true if hardware configuration is enabled and false if not.
Returns
EI_API_ADP_EError_t as uint32_t value.
Return values
EI_API_ADP_eERR_OKSuccess.
EI_API_ADP_eERR_GENERALGeneral error.
Example
#include <api/EI_API.h>
EI_API_ADP_T* pEI_API_ADP = NULL;
uint32_t errCode;
uint8_t numInterfaces = 1;
bool hwConfigEnabled;
pEI_API_ADP = EI_API_ADP_new(numInterfaces);
errCode = EI_API_ADP_isHwSettingEnabled(pEI_API_ADP, &hwConfigEnabled);
