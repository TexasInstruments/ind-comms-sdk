Setup

Software installation

  1. Install Code Composer Studio for Sitara(TM) processors (CCS).
  2. Install System Configuration Tool (sysconfig).
  3. Install ArmÂ® code generation tools - compiler (ARM-CGT)
  1. Install Industrial Communication Software Development Kit (IND-COMMS-SDK) AM243X with demos and device description ESI files

for details see Getting started

Connect the development board

AM243x-LP

Board details: LP-AM243

  1. Setup a dip switches on the board : DEV-BOOT
DEV-BOOT Mode

  1. Connect the evaluation board to your computer using the micro-USB cable.

    On the LP bards over the USB cable are connected the JTAG interface (XDS110) and the UART console.

USB connector on the AM243x LP Board
  1. Connect an ethernet cable from a PC running the TwinCAT to EtherCAT IN/Port0 of the PRU-ICSS on the development board.
EtherCAT connectors on the AM243x LP Board
  1. If you have multiple devices in chain, please connect from EtherCAT OUT/Port1 to IN/Port0 of next device. For the last device in chain, OUT/Port1 will be left open (if NOT using redundancy mode).
  2. Connect a USB Type-C Power Supply to the board
Board power connector on the AM243x LP

Steps to Run the Example

  1. When using CCS project to build, import the CCS project for the required combination and build it using the CCS project menu, see Using SDK with CCS Projects.
  2. When using makefiles to build, note the required combination and build using make command, see Using SDK with Makefiles.
  3. Lauch a CCS debug session and run the executable, see CCS Lauch, Load and Run.
