Release Notes

Overview

The Sitara IO-Link master stack can be quickly integrated into supported devices
and helps to create IO-Link master applications on TI Sitara Embedded Processors with PRU-ICSS.

Release Version 1.07.00

New Features:

  • [IOL8M-195] Test on AM243x EVM
    AM243x EVM example has been successfully tested on AM243x EVM hardware.
  • [IOL8M-958] 400 µs Cycle Time
    Adjusted the IO-Link master to support IO-Link devices with a 400 µs cycle time.
    Currently, only one device with a 400 µs cycle time can be guaranteed; further
    testing will be required to verify that more than one device with a cycle time of
    400 µs can be supported.
  • [IMS-35] Wrong error code (IoLError_ServiceTempUnavailable)
    When the port was correctly configured to output, the stack generated an incorrect
    error code (IoLError_ServiceTempUnavailable). The corrected code is now
    generated: IOLM_SMI_ePortStatus_DO_DQ.

Release Version 1.01.00

New Features:

  • Standards supported by this Release are:
    IO-Link Stack according to IEC 61131-9 Spec V1.1
    IODD V1.1
  • Functions supported by this Release are:
    IO-Link Master Stack
    Standard Master Interface
  • Communication Classes supported are:
    Com1, Com2, Com3 (4.8, 38.4, 230.4 Kbps)
  • Telegram Types supported are:
    0, 1, 2.1, 2.2, 2.3, 2.4, 2.5
  • This Release supports up to 8 IO-Link Channels in parallel.
