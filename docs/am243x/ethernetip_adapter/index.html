EtherNet/IP™ Adapter3.07.01
Introduction

Scope

This software component is designed for Texas Instruments SITARA AM243X based platforms with PRU-ICSS IP to enable customers to integrate EtherNet/IP™ Adapter functionality into their system. It implements a CIP™ (Common Industrial Protocol, see THE CIP NETWORKS LIBRARY – Volume 1) and EtherNet/IP (EtherNet/IP Adaptation of CIP, see THE CIP NETWORKS LIBRARY – Volume 2) stack based on the PRU-ICSS firmware by Texas Instruments. The supported CIP and EtherNet/IP functionality is listed in the Datasheet below. The EtherNet/IP stack is executed on a dedicated R5F core running FreeRTOS.
